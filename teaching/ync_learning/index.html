Description:

Machine learning is a collection of techniques where computers can learn from data, without being explicitly programmed. For instance, when we train a program using human-face image data, it should be able to locate any faces in a given image; yet, if we train the same program using flower image data, it should be able to locate flowers in a given image, without changing the program itself. To accomplish this, statistical machine learning, which relies heavily on probabilistic and statistical analysis is required. This module will discuss the recent techniques in machine learning including both shallow learning and deep learning.

Prerequisite: Programming skill in either python, c/c++, or matlab.
Textbook: "Pattern Recognition and Machine Learning", by Christopher Bishop.
