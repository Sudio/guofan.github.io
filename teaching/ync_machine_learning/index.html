Description:

Machine learning is about how to make machines/computers identify patterns from data, extract the patterns, and based on them, make correct inference or prediction automatically. These capabilities are scientifically important and the core of artificial intelligence; namely, to enable machines to learn without being explicitly programmed using specific and fixed rules. Machine learning is practically useful, since nowadays we are bombarded with a huge number of various data from various sources. We hope it can make sense of this seemingly random huge data.

Technically, there are at least three types of learning: supervised, unsupervised, and reinforcement learning. Supervised learning means a machine is trained using both data and their annotations. Usually, the annotations are provided by us (humans). Unsupervised learning means a machine is trained using only raw data, without any manual annotations. In this case, the machine will try to find and extract the most significant patterns from the data by itself. Reinforcement learning means a machine is trained using data and feedback/rewards from its actions. Thus, there are some interactions between the machine and its environments.

This course focuses on these three types of learning, including shallow learning and deep learning. It should interest students who want to study/work in big data, AI (artificial intelligence), and data science.
