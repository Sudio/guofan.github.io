YSC3203 Advanced Algorithms and Data Structures

Semester 1, 2016/2017
Yale-NUS College





Description:

This module is about the fundamental principles of algorithm analysis and design. Generally, the goal of designing an algorithm is to process input data in order to generate desired output. However, we are bound to create algorithms with running time as efficient as possible, surely without sacrificing the correctness or the other requirements of our desired output. To know the efficiency of algorithms requires running time analyses or proofs, including those that utilize recursive and random processes. As for algorithm design, we study some basic techniques that are widely used, such as, randomized algorithms, divide and conquer, dynamic programming, greedy algorithms, graph algorithms, linear programming, parallel programming, etc. Aside from algorithms, we also discuss data structures, particularly dynamic data structures and their access time analysis: hashing, randomized binary search trees, black-red trees, augmenting data structures and an access-time analysis called amortized analysis. This course will be interesting (and perhaps also exciting), since we will see many elegant and beautiful algorithms/analysis.
